About Me

Hi, I'm Raph. I'm a developer interested in making software that's useful to me and to others. I'm very interested in learning more web development and low level system programming. (Keeping my eye on embedded for later).

I started programming back in 2017, when I had just switched to linux and over time I started scripting things and making tiny utilities for myself. Eventually I went deeper and deeper into it and ended up loving it.

I'm kinda really into using very efficient programs, so I want to be able to do efficient software myself. You can find me and see what I'm up to lately on github or the other links on the top of the page.
